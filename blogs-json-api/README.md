# JavaScript Blog Practice with JSON API

A simple blog application built using **HTML**, **Tailwind CSS (CDN)**, and **Vanilla JavaScript** to practice API fetching, async/await, and dynamic UI rendering.

---

## Technologies Used
- HTML5
- Tailwind CSS (CDN)
- JavaScript (ES6+)
- Async / Await
- Fetch API
- JSON API

---

## Features
- Fetch blog posts from a JSON API
- Display blogs dynamically
- Error handling for failed API requests
- Clean and responsive UI using Tailwind CSS

---

## Purpose
This project is created for learning and practicing:
- JavaScript async/await
- API integration
- Clean and readable code structure
- Frontend logic building

---

## Notes
This is a practice project and does not include a backend or database.
